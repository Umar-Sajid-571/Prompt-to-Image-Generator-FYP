## ğŸ“– Project Overview
This project bridges the gap between Natural Language Processing (NLP) and Computer Vision. It leverages the **Stable Diffusion v1.5** architecture to allow users to generate custom artwork, realistic photos, or conceptual designs through a simple chat interface.

### âœ¨ Key Features:
- **Modern Chat UI:** Interactive and intuitive chat-style flow for prompt input.
- **Customizable Inference:** Users can adjust **Inference Steps** and **Guidance Scale** from the sidebar to control image quality and creativity.
- **Resolution Control:** Supports multiple resolutions from 64x64 up to 512x512.
- **Session History:** Keeps track of generated images during the current session.
- **Instant Download:** One-click button to save generated images in PNG format.
